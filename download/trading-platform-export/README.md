# منصة التداول الشاملة - Trading Platform

## 🚀 تشغيل المشروع

### 1. تثبيت المتطلبات
```bash
npm install
```

### 2. إعداد قاعدة البيانات
```bash
npx prisma generate
npx prisma db push
```

### 3. تشغيل المشروع
```bash
npm run dev
```

### 4. فتح المتصفح
```
http://localhost:3000
```

## 📁 هيكل المشروع

```
src/
├── app/
│   ├── api/           # API Routes
│   ├── page.tsx       # الصفحة الرئيسية
│   └── layout.tsx     # التخطيط الرئيسي
├── components/
│   ├── trading/       # مكونات التداول
│   └── ui/            # مكونات الواجهة
├── lib/               # المكتبات والأدوات
├── locales/           # ملفات الترجمة
└── hooks/             # React Hooks

prisma/
└── schema.prisma      # هيكل قاعدة البيانات
```
